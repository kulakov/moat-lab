# Канвас 2. Фильтр фейк-рвов

Прогони каждую выжившую гипотезу через протокол. Роли: **автор защищается**, **другой фаундер + скептик атакуют**, **фасилитатор выносит вердикт** (не автор).

## Протокол копирования (на каждую гипотезу)

| Проверка | Вопрос | Прошла? |
|---|---|---|
| **Тест копирования** | Конкурент с бюджетом и ИИ догонит за квартал? | ☐ да → фейк / ☐ нет |
| **Pricing power** | Клиент платит премию И говорит «никто не делает как вы»? | ☐ |
| **Retention** | Удержание держится без скидок? (для switching) | ☐ |
| **Барьер ≠ бенефит** | Это барьер, а не просто быстрый рост? | ☐ |
| **Не фейк §фейки** | Не фича / generic-данные / модель / временная фора? | ☐ |
