# Канвас 0. Карта продукта

Заполни до сессии. Правило: каждое поле — **факт или явное «неизвестно»**, не догадка под видом факта. Пустое поле — сигнал «здесь рва нет / не знаем».

| # | Поле | Твой ответ |
|---|---|---|
| 1 | **Однострочник** (что, кому, какую работу клиента) | |
| 2 | **Сегмент и ЛПР** (кто платит / решает / пользуется) | |
| 3 | **Статус-кво** (чем клиент решает задачу сейчас, включая «никак» и Excel) | |
| 4 | **Что копируется за квартал** (честно: какие фичи повторит конкурент с ИИ) | |
| 5 | **Что накапливается** (данные / связи / контент / операционная сложность со временем) | |
| 6 | **Издержки ухода** (что клиенту больно/дорого бросить) | |
| 7 | **Эксклюзивный доступ** (редкие данные, регуляторика, талант, IP, партнёрства) | |
| 8 | **Каналы + CAC** (где берёшь клиентов, почём по каждому) | |
| 9 | **Метрики** (retention/плато, NDR, доля органики, динамика CAC, реакция на рост цены) — числами или «нет данных» | |
| 10 | **Что инкумбент не повторит** (модель/прайсинг, бьющий по его выручке) | |
| 11 | **Наивная гипотеза рва** (где ты *думаешь*, что защищён) | |

**Стадия (отметь):** ☐ pre-product (нет выручки/метрик) ☐ post-revenue

**Gate перед сдачей:** ☐ поле 4 заполнено честно ☐ поле 11 названо ☐ метрики числами или «нет данных» ☐ дыры помечены «неизвестно»

> Privacy: карта уходит в модель для подготовки. Не вписывай то, что не готов раскрыть.
