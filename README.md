# Лаборатория «Ров» — кит для команд

Метод и инструменты, чтобы найти и проверить **ров** вашего стартапа — то, что не скопировать за два часа в эпоху ИИ. Запускается в [Claude Code](https://claude.com/claude-code): открываете репозиторий, и Claude ведёт вас через 6 этапов.

## Что это

Найти проблему и собрать под неё софт с ИИ сегодня умеет каждый — это пара часов. Поэтому защищает не функциональность, а **ров**: то, что накапливается и встраивается. Этот кит помогает за полдня:

1. **отсеять фейк-рвы** (то, что копируется за квартал);
2. **построить причинную модель**, где время работает на вас — разрыв с конкурентом растёт, а не схлопывается;
3. **запустить внешнюю проверку** с ранними метриками и зафиксированными порогами.

> Честно: за полдня ров не доказать — он подтверждается только устойчивостью во времени. Кит помогает **опровергнуть плохие варианты**, **смоделировать** рабочий и **запустить проверку**, вердикт по которой приходит уже по данным.

## Как пользоваться

1. Установите [Claude Code](https://claude.com/claude-code).
2. Склонируйте репозиторий и откройте его в Claude Code:
   ```bash
   git clone <этот-репо> moat-lab && cd moat-lab
   claude
   ```
3. Скажите Claude: **«Помоги найти мой ров»** — он откроет карту продукта и поведёт по этапам.

Claude читает `CLAUDE.md` (его инструкции) и использует скилы из `.claude/skills/`.

## Структура

```
CLAUDE.md            — инструкции Claude: метод, роль, поток этапов
REFERENCE.md         — типология рвов, метрики, фейки
PROGRAM.md           — программа сессии (run-of-show) на полдня
canvases/            — канвасы, которые вы заполняете на каждом этапе
.claude/skills/      — 14 скилов (ИИ-цепочки) по этапам
```

## Принцип

Claude здесь — **фасилитатор и адвокат дьявола**, а не оракул. Он генерирует черновики и контраргументы, но **вердикт всегда выносите вы**. Его задача — найти, где ваш ров копируется за квартал, а не похвалить идею.
